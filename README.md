## Instalation GUIDE

- Navigate to src folder
    - `cd /src`

- Make sure the script is executable
    - `chmod +x execute_sql.sh`

- spin up the containers
    - `docker-compose up -d`

- Configure the database
    - `./execute_sql.sh`
